Understanding STL: vector, set, map

December 1, 2025
7 min read
Nadim
STL C++ Data Structures

The C++ Standard Template Library (STL) provides ready-to-use, highly optimized data structures and algorithms. In this guide, we will explore three essential STL containers: vector, set, and map. Understanding them is crucial for efficient competitive programming and software development.

What is STL?

STL is a collection of template classes and functions that help manage data and perform operations efficiently. It reduces coding time and increases code readability while maintaining performance.

Key Insight

STL provides generic implementations of common data structures and algorithms, letting you focus on solving problems.

Vector

A vector is a dynamic array that can resize itself automatically. It provides random access and is efficient for append operations.

Basic Operations

vector<int> v;
v.push_back(10); // Add element
v.push_back(20);
cout << v[0]; // Access element
v.pop_back(); // Remove last element
cout << v.size(); // Number of elements

Use Case

Vectors are ideal when you need a resizable array with fast access by index.

Set

A set stores unique elements in sorted order. It is useful for fast lookups, insertions, and deletions.

Basic Operations

set<int> s;
s.insert(5);
s.insert(2);
s.insert(5); // Duplicate ignored
cout << s.count(2); // Check existence
s.erase(2); // Remove element

Use Case

Sets are perfect for storing unique items and performing quick searches.

Map

A map stores key-value pairs, where each key is unique. Maps allow fast lookups, insertions, and updates based on keys.

Basic Operations

map<string, int> mp;
mp["Alice"] = 25;
mp["Bob"] = 30;
cout << mp["Alice"]; // Access value
mp.erase("Bob"); // Remove key-value pair

Use Case

Maps are useful when you need to associate values with keys and retrieve them efficiently.

Mastering vector, set, and map is essential for efficient coding in C++. They simplify complex operations and improve both runtime and readability. Practice implementing them in different problems to strengthen your understanding.
